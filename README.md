# Fake News Detection on Twitter

This project detects fake news on Twitter using Machine Learning.

## Features
- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression model
- Streamlit UI
- Confidence score & reasoning

## Tech Stack
- Python
- Scikit-learn
- Streamlit

## How to Run
streamlit run app.py
